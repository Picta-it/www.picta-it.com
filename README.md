# Picta-it
Picta-it Website. Based upon [sails](https://github.com/balderdashy/sails) framework.

To clone it :
```sh
git clone git@github.com:Picta-it/www.picta-it.com.git
```

To install it :
```sh
# Install client side dependencies
bower install
# Install server side dependencies
npm install
# Updated bootstrap less directory
npm run-script link-css
```
